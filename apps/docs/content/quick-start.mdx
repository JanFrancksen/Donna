# 🧪 Full Stack Monorepo Starter

Welcome to the official documentation for the **Full Stack Starter Monorepo** — a developer-focused, production-ready starter kit for building modern web and mobile applications with shared UI, auth, and infrastructure.

---

## 🚀 What's Included

This starter gives you a powerful full-stack setup out of the box:

- **Web App**: Next.js (App Router)
- **Mobile App**: Expo + React Native
- **Database**: Drizzle ORM + SQLite/Postgres
- **Authentication**: Better Auth
- **Payments**: Polar.sh
- **API & Workers**: Hono + Trigger.dev
- **Shared UI Library**: TailwindCSS + TypeScript + Radix
- **Validation**: Zod
- **Documentation**: Fumadocs
- **Analytics**: Built-in tracking with [your tool]
- **Monorepo Tooling**: Turborepo (or Nx), pnpm workspaces
- **Emails**: System emails with Resend/Postmark
- **Typed Everything**: End-to-end type safety via TypeScript
